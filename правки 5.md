# ТЗ: Добавление поля "Закупочная цена" в раздел "Склад"

## Контекст
Пользователь сообщил, что в разделе управления складом не хватает возможности указывать и видеть закупочную стоимость товаров. Сейчас там есть только наименование, артикул и (вероятно) цена продажи/количество.

## Задача: Интеграция закупочной цены
Внеси следующие изменения в код логики склада:

1. **Обновление UI (Форма добавления/редактирования товара на складе):**
   - Добавь новое обязательное поле ввода `Закупочная цена` (тип: number, минимальное значение: 0).
   - Расположи его логично — рядом с полем розничной цены (если оно есть) или количеством.

2. **Обновление UI (Таблица/Список товаров на складе):**
   - Добавь новую колонку или блок данных `Закупка` (или `Закупочная цена`) в общий список товаров на складе, чтобы пользователь сразу видел стоимость закупки без необходимости открывать карточку товара.

3. **Обновление БД и Backend-логики (Сохранение/Загрузка):**
   - Обнови схему сохранения данных склада (БД/JSON/модель): добавь новый параметр `purchase_price` (закупочная цена).
   - Убедись, что при создании нового товара и при редактировании существующего это значение корректно записывается в базу и считывается из неё.
