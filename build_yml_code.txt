GitHub Actions workflow для сборки .exe файла.

Файл размещается в: .github/workflows/build.yml

Сборка запускается:
  - При пуше тега v* (например v1.0.0)
  - Вручную через Actions -> Build EXE -> Run workflow

Результат: OrdersTable.exe в разделе Artifacts.

---

name: Build EXE

on:
  push:
    tags:
      - 'v*'
  workflow_dispatch:

jobs:
  build-windows:
    runs-on: windows-latest
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt
          pip install pyinstaller

      - name: Build EXE
        run: pyinstaller build.spec

      - name: Upload artifact
        uses: actions/upload-artifact@v4
        with:
          name: OrdersTable-windows
          path: dist/OrdersTable.exe
